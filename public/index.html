✋ Donner un coup de pouce

Voici comment nous soutenir activement :

  1. Décrivez la modification que vous aimeriez voir sur le site (un bug, une amélioration, une nouvelle fonctionnalité technique).
  2. Choisissez la nature de votre demande parmi les 3 catégories ci-dessous — chacune fixe le don minimum.
  3. Effectuez votre don via l'un des réseaux (M-Pesa, Airtel, Orange, Africell) au numéro affiché, pour un montant au moins égal au minimum de la catégorie choisie.
  4. Indiquez le montant envoyé et la référence de la transaction (le numéro/ID de confirmation reçu par SMS).
  5. Envoyez votre requête. Un code unique s'affiche : conservez-le, il vous permettra plus tard de confirmer que la modification a été vue et faite.

Une fois envoyée, votre requête apparaît sur la page de suivi. Dès que le développeur réalise la modification, le 1er accusé ✓✓ s'active. Quand vous constatez le changement, utilisez votre code pour activer le 2e accusé ✓✓ (bleu) = demande confirmée.
